# Security Policy

## Reporting a vulnerability

Please do not disclose security vulnerabilities in public issues. Contact the
project maintainers privately with a clear reproduction and impact summary.

Helikon does not collect telemetry, account tokens, or hardware identifiers.
Configuration imports are treated as untrusted data and must never execute code.

The current release-candidate review scope, controls, residual risks, and
release gate are documented in [docs/security-review.md](docs/security-review.md).
